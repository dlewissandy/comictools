# Comic Creator App

Welcome to the Comic Creator App! This application allows you to create and customize your own comics using a user-friendly interface.

## Features

- Create custom comic panels
- Add text, images, and effects
- Save and share your comics

## Installation

To install the required dependencies, run:

```
uv pip install -r requirements.txt
```

## Running the App

To start the comic creator app, run the following command:

```
uv run streamlit run app.py
```

This will launch the Streamlit web interface in your browser.
